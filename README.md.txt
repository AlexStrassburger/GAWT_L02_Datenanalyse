# GAWT – Projekt Datenanalyse

## Beschreibung
Dieses Repository enthält die Projektarbeit im Modul
„Grundlagen und Anwendungen der Wahrscheinlichkeitstheorie (GAWT)“.

Analysiert werden mehrere Datensätze mit statistischen Methoden.
Die Auswertung erfolgt in Jupyter Notebooks, die Ergebnisse
werden in einem Word-Bericht dokumentiert.

## Projektstruktur
/data       – Roh- und bereinigte Datensätze   
/notebooks  – Jupyter Notebooks zur Analyse  
/figures    – Exportierte Grafiken für den Bericht  
/bericht    – Word-Dokumente der Projektarbeit
/Export     - Für Rang und Urlisten der Datensätze 1 und 2  

## Verwendete Werkzeuge
- Python (pandas, numpy, matplotlib)
- Jupyter Notebook
- Git & GitHub

## Zusammenarbeit
Dieses Projekt wird im Team bearbeitet.
Änderungen werden versioniert und über GitHub synchronisiert.
